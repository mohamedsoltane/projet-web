MS Location

Renseignements sur le projet

  • Catégorie: Web & Mobile
  • Projet date: Décembre, 2022
  • Projet URL: www.ms-location.tn

Notre tâche consiste à mettre en place une application desktop, web et mobile qui permet l’amélioration des services pour chaque client. Deplus, celle-ci doit notamment permettre la gestion des stocks, commandes des clients, ressource humaine de l’entrepôt ainsi que la gestion des réclamations de chaque client.

    Les technologies utilisées :

  • Java Script
  • CSS
  • Bootstrap
  • Html
